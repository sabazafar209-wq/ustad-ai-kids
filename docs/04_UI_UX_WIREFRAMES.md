# 1. Design Philosophy

The interface should feel like a friendly teacher rather than a traditional educational application.

The design should inspire:

- Curiosity
- Confidence
- Exploration
- Safety
- Fun
- Simplicity

Every screen should minimize distractions and maximize learning.

---

# 2. Design Principles

### Simplicity

Children should never be confused.

---

### Consistency

Buttons, icons, and navigation should remain consistent throughout the application.

---

### Positive Reinforcement

Celebrate effort instead of perfection.

---

### Child Safety

No advertisements.

No external links.

No social interaction.

---

### Parent Transparency

Parents should always understand what their child is learning.

# UI Principles

Every screen should answer three questions immediately:

1. Where am I?
2. What should I do next?
3. How am I progressing?

If a child cannot answer those questions within a few seconds, the interface should be simplified.
