# Purpose

This document defines the complete visual design language for the application to ensure consistency across all screens.

---

# Design Philosophy

The interface should feel like:

- Friendly
- Calm
- Educational
- Modern
- Safe
- Playful

Never childish.

Children should feel respected.

Parents should feel confident.
