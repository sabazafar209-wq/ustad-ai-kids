# 1. Curriculum Vision

The curriculum is designed to transform children from curious beginners into confident problem solvers who understand programming, computational thinking, and Artificial Intelligence through interactive, personalized learning.

The emphasis is on understanding concepts rather than memorizing facts.

---

# 2. Educational Objectives

By completing the curriculum, learners should be able to:

- Think logically
- Solve simple problems
- Understand algorithms
- Build confidence using technology
- Write beginner-friendly code
- Understand how AI works
- Use AI responsibly
- Create simple AI-powered projects
